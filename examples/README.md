# Examples

Toy inputs for trying Hydra without setting up a real CTF target.

| File | Purpose |
|---|---|
| `challenges-toy.json` | Three trivial challenges (rot13, base64, strings) that exercise the full pipeline in ~45s |
| `challenges-bug-bounty-template.json` | Template for bug-bounty workflows. Pair with `prompts/bb-babysit.md` |
| `quickstart.md` | 5-minute walkthrough of what running Hydra actually looks like |

For real CTF batches, write your own `challenges.json` — see the
[Input format](../README.md#input-format) section of the main README.
